Drop the same action on multiple keys to spread one album cover across them. For a 3 x 3 cover wall, use 9 keys with grid size 3 and pick row 0–2 / column 0–2 for each. Each key can do its own thing on press – mix and match next, previous, volume, mute. The page actions only work inside the plugin's bundled "Covers 1–4" profiles (Stream Deck doesn't let plugins flip pages of personal profiles) – set "On page" to the Covers profile this key lives in.
