# ScreenSaver

ScreenSaver is a local-only Windows 10/11 desktop suite that tiles a music library's album covers into a full-screen idle mosaic, floats the now-playing track as a focal cover, and mirrors that cover to the desktop wallpaper, the lock screen, a LAN web page, and a Stream Deck. It serves a single operator on one machine, with no cloud dependency. For what it is, how it is built, and how it deploys, see [docs/index.html](docs/index.html).
